# 📝 To-Do List App

A simple and interactive **To-Do List web application** built using **JavaScript, DOM manipulation, and Bootstrap**.  
This project helps users manage their daily tasks efficiently with a clean, responsive, and modern UI.

---

## 🚀 Live Demo

## 🔗 [Go_ToDo](https://mostafa-younis7.github.io/To-Do-List/)

## 📌 Features

- ➕ Add new tasks easily
- 📋 Display tasks dynamically
- ❌ Delete tasks instantly
- 🎨 Clean and responsive UI using Bootstrap
- ⚡ Fast performance with pure JavaScript

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the application
- **CSS3** – Custom styling
- **Bootstrap** – Responsive design and UI components
- **JavaScript (ES6)** – Application logic
- **DOM Manipulation** – Dynamic rendering
- **Array Methods** – Efficient task management
